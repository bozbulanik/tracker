Activity tracker for my personal use. 

What it tracks right now:

* Mouse click (left, right, middle)
* Keyboard press (key statistics)
* Mouse movement (navigation, scroll)
* Application usage [WIP]
* File activity [WIP]
* Sound/media usage [WIP]
